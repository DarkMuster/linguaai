## ✨ Features

| Feature | Details |
|---|---|
| 🌍 **7 Languages** | English, Japanese, German, Spanish, French, Russian, Arabic |
| 📊 **CEFR Levels** | A1 → A2 → B1 → B2 → C1 → C2 (tab navigation) |
| 🎮 **Exercise Types** | Multiple choice · Pair matching · Fill-in-the-blank |
| ❤️ **Game Mechanics** | Hearts · XP · Daily Streaks · Stars · Confetti |
| 🌸 **Sakura Canvas** | Animated falling petals + floating language characters |
| 💰 **Monetisation** | Google AdSense slots + Stripe Pro paywall |
| 📱 **Responsive** | Mobile-first, works on all screen sizes |
| 💾 **Offline-ready** | All progress saved in localStorage |

## 🚀 Deploy (free)

### GitHub Pages (you're already here!)
1. Go to **Settings → Pages**
2. Source: **Deploy from branch → main**
3. Save → live at `https://darkmuster.github.io/linguaai`

### Netlify Drop
1. Go to **[netlify.com/drop](https://netlify.com/drop)**
2. Drag `index.html` → instant live URL

---

## 💰 Monetisation Setup

### 1. Google AdSense
Uncomment the AdSense `<script>` tag in `index.html` `<head>` and replace `YOUR_ID`.

### 2. Stripe Premium ($4.99/month)
Find `STRIPE_LINK` near the top of the `<script>` section in `index.html` and replace with your Stripe Payment Link.

## 🎨 Design System

**Sakura Classic** — inspired by traditional Japanese aesthetics:
- **Font**: Noto Serif JP (headings) + Inter (body)  
- **Palette**: Deep ink `#120808` · Sakura `#D4627A` · Gold `#C9993A`
- **Animation**: Canvas falling cherry blossom petals + floating multilingual characters
